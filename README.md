# Stream - SNSU Community Forum App


### Overview

Stream is a community-driven forum platform built fot **Surigao Del Norte State Uiversity (SNSU).** It serves as a modern digital space where students and faculty can connect, share, ask questions and engage in meaningful discussions across various academic and campus communities.
